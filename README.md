# Phishing Email Detection Model

## Overview

A Machine Learning project built with Scikit-learn that classifies emails as:

- Phishing
- Safe

The model uses TF-IDF text feature extraction and a Naive Bayes classifier.

---

## Features

- Email text analysis
- TF-IDF feature extraction
- Phishing classification
- Accuracy calculation
- Confusion Matrix generation
- User email scanning

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python phishing_detector.py
```
